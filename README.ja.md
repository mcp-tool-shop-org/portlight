貿易を重視した海洋戦略ゲーム。ルートの最適化、契約、インフラ、金融、評判など、様々な要素を駆使して、5つの地域で商人のキャリアを築き上げましょう。すべてはターミナルから。

## インストール

```bash
pip install portlight
```

Pythonがインストールされていませんか？代わりにnpmのパッケージを使用してください。

```bash
npx @mcptoolshop/portlight
```

## Portlightを選ぶ理由

多くの交易ゲームでは、交易は単に数値が上昇するだけのものとして扱われます。Portlightでは、交易を商業的な活動として捉えています。

- **価格はあなたの取引によって変動します。** 穀物を港で販売すると価格が暴落します。すべての販売が地域の市場に影響を与えます。
- **各港には独自の経済的特徴があります。** ポルトノボは安価な穀物を生産します。シルクヘイブンは大量の絹を輸出します。これらは構造的なもので、ランダムなものではありません。
- **航海にはリスクが伴います。** 嵐、海賊、検査、季節的な危険などがあります。積載物、船体、そして乗組員の質が重要です。
- **契約には証拠が必要です。** 期限内に、正しい商品を正しい港に届けましょう。商品の出所は追跡されます。
- **インフラは取引方法を変えます。** 倉庫は貨物を一時的に保管します。仲介人は契約を改善します。ライセンスは特別なアクセス権を付与します。
- **評判はチャンスを広げたり、閉ざしたりします。** 商業的な信頼、税関の取り締まり、地域の評価、そして裏社会とのコネクション。これら4つの要素が、あなたが何ができるか、どこで活動できるかを左右します。
- **ゲームはあなたが築き上げたものを読み取ります。** あなたの取引履歴、インフラ、評判、そして航路は、あなたのキャリアを形成します。あなたがどのような商人になったかに基づいて、4つの異なる勝利ルートが存在します。

## 世界

5つの地域、20の港、43の航路。そして、生きているかのような経済。

| 地域 | 港 | キャラクター |
|--------|-------|-----------|
| **Mediterranean** | ポルトノボ、アル＝マナール、シルバベイ、コルセアーズ・レスト | 穀物、木材、スパイスの市場。安全な出発地点。 |
| **North Atlantic** | アイアンヘイブン、ストームウォール、ソーンポート | 鉄、武器、軍需品。厳格な検査。 |
| **West Africa** | サンハーバー、パームコーブ、アイアンポイント、パールシャロウズ | 綿、ラム酒、真珠。最も安価な物資。 |
| **East Indies** | ジェイドポート、モンソンリーチ、シルクヘイブン、クロスウィンドアイル、ドラゴンズゲート、スパイスナローズ | 絹、スパイス、陶器、お茶。最も高い利益率。モンスーンのリスク。 |
| **South Seas** | エンバーアイル、タイフーンアンカレッジ、コーラルスローン | 真珠、医薬品。遠くの、ゲーム終盤の水域。 |

各港に134人の名前付きNPCが存在します。異なる水域を支配する4つの海賊勢力。季節によって変化する天候が、危険と需要を変化させます。祭り、迷信、そして乗組員の士気を反映した文化要素も存在します。

## 9人の船長

| 船長 | ホーム | 利点 | 欠点 |
|---------|------|------|-----------|
| **Merchant** | ポルトノボ | より良い価格、信頼が急速に向上 | 税関の取り締まりペナルティが2倍 |
| **Smuggler** | コルセアーズ・レスト | 闇市場、違法取引 | リスクが高い、検査の頻度が高い |
| **Navigator** | モンソンリーチ | より速い船、より長い航続距離 | 初期の評価が低い |
| **Privateer** | アイアンヘイブン | 海戦、強襲の有利 | 商人の評判が低い |
| **Corsair** | コルセアーズ・レスト | 戦闘と取引のバランス | 万能ではない |
| **Scholar** | ジェイドポート | 情報優位、より良い契約 | 少ない初期資金、脆弱 |
| **Merchant Prince** | ポルトノボ | 多い初期資金、特別なアクセス権 | 高い手数料、海賊の標的 |
| **Dockhand** | クロスウィンドアイル | 最も安い乗組員、手際が良い | 最も少ない初期資金 |
| **Bounty Hunter** | ストームウォール | 戦闘の腕前、勢力との関係 | 価格が低い、信頼されていない |

各船長は異なる港からスタートし、異なる契約に遭遇し、異なる勝利ルートを目指します。ゲームはあなたを固定するのではなく、あなたが何をするかを観察し、あなたがどのような商人になったかを教えてくれます。

## ゲームの流れ

```
Inspect market → Buy cargo → Sail → Sell → Reinvest → Build access → Pursue destiny
```

## 始め方

```bash
portlight new "Captain Hawk" --type merchant
portlight market
portlight buy grain 10
portlight routes
portlight sail al_manar
portlight advance
portlight sell grain 10
portlight milestones
```

最初のセッションをガイドするチュートリアルは、[docs/START_HERE.md](docs/START_HERE.md) を参照してください。

## システム

**経済**：20の港、18の商品、43の航路において、需給バランスに基づいた価格設定。過剰な供給に対するペナルティ。市場の変動は機会を生み出す。地域ごとの需要変動により、各港の輸入・輸出の特性が明確になる。

**航海**：数日間の航海。天候、海賊との遭遇、検査が発生。食料は毎日消費される。船体は損傷を受ける。乗組員の士気が変動する。季節によって危険な海域が変わり、安全な航路が変化する。

**契約**：信頼と実績によって結びついた6つの組織。調達、供給不足の緩和、高級品取引、積み荷の返却、定期航路、組織からの委託など。現実的な納期と、それに伴う結果。

**評判**：4つの要素：地域での地位、商業的な信頼、税関の監視、裏社会との繋がり。高い信頼性は、より有利な契約を可能にする。高い監視レベルは、検査や港への入港拒否を引き起こす。各船長は、異なる倫理観に基づいた経済活動を行う。

**戦闘**：7種類の近接武器、7種類の遠距離武器、地域ごとの戦闘スタイルを用いた、詳細な個人戦闘（攻撃のタイプ：突き/斬り/防御）。船同士の戦闘では、乗り込みや大砲が使用可能。短く、激しく、そして重大な結果をもたらす。

**海賊組織**：クリムゾン・タイド（地中海）、アイアン・ウルブズ（北大西洋）、ディープ・リーフ・ブラザーフッド（南洋）、モンソン・シンジケート（東インド）。各組織は、領土、好みの商品、名前のついた船長、そしてあなたに対する態度を持っている。

**インフラ**：倉庫（3段階）、仲介事務所、購入可能なライセンス5種類。維持費が発生する。これらは、貿易のタイミング、規模、アクセスを変化させる。

**金融**：保険（船体、積荷、契約保証）と、金利付きのクレジット（3段階）。リスクとリターンのバランス。

**仲間**：5つの士官役職（水兵、航海士、外科医、密輸業者、船大工）。個性、士気、そして離脱条件を持つ、名前のついた仲間たち。

**キャリア**：6つの組織に属する、27の目標。13種類のキャリアプロファイル。4つの勝利条件：法的な貿易会社、秘密結社、広大な海洋支配、商業帝国。

## 勝利への道筋

- **合法的な貿易会社** — 厳格な正当性。高い信頼性、高額な契約、清廉な評判、広範なインフラ。
- **影のネットワーク** — 秘密裏で利益の高い取引。監視下にある高利益率、リスク管理、強靭な運営。
- **広大な海洋ルート** — 長距離の商業力。東インドへのアクセス、遠隔地のインフラ、主要ルートの支配。
- **商業帝国** — 統合された多地域運営。すべての地域にインフラを構築、多様な収入源、金融力。

## ボードゲーム（印刷して遊ぶ形式）

完全なボードゲーム版を生成：カード、ボード、ルールブック、スコアトラック。

```bash
pip install portlight[printandplay]
portlight print-and-play
```

2～4人のプレイヤーが参加する、競争型の商人アドベンチャー（約90分）。非対称な船長、契約競争、そして評判/監視の緊張感が特徴。詳細なルールは、[docs/PRINT_AND_PLAY_RULES.md](docs/PRINT_AND_PLAY_RULES.md) を参照。

## 品質

- 72以上のファイルにまたがる1,832件のテスト
- 9つの複合的なストレスシナリオ下で、クロスシステムの一貫性を14項目で検証
- バランス調整：7つのシナリオパックにまたがる7つのポリシーボット
- セーブデータ形式 v12。完全な移行チェーンをサポート
- コードは、Ruffでクリーン化されており、Python 3.11/3.12/3.13に対応

## セキュリティ

ローカル環境でのみ動作するコマンドラインゲーム。ゲームプレイ中にネットワーク接続は一切使用しない。セーブデータは、ローカルファイルシステム上の `saves/` および `artifacts/` フォルダにJSON形式で保存される。機密情報やテレメトリーは一切含まれておらず、管理者権限も不要。詳細は、[SECURITY.md](SECURITY.md) を参照。
